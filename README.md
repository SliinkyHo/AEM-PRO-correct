# AEM Pro Pressure Washing — Hero Concept

A responsive, single-page hero concept for AEM Pro Pressure Washing.

## Files

- `index.html` — the complete page
- `assets/` — logo and imagery required by the page
- `preview.png` — a static preview of the concept

## View locally

Open `index.html` in a web browser. Keep the `assets` folder in the same directory as `index.html`, or the logo and hero image will not load.

## Publish

Upload `index.html`, the full `assets` folder, and this README to the repository. Once GitHub Pages is enabled, the site can be shared as a normal web link.
